7 AI-агентов для полного цикла подбора

Каждый агент отвечает за свой этап рекрутинга — от анализа вакансии до адаптации сотрудника

Агент 1: Профайлер

Анализирует вакансию и создаёт портрет идеального кандидата

Результат: Чек-лист: кто нужен (пол, возраст, навыки, где искать)
Агент 2: Копирайтер

Пишет продающее объявление для публикации

Результат: Текст для HH.ru / Avito / WorkHere (заголовок + 5 буллетов + призыв)
Агент 3: Скриптер

Создаёт сценарий телефонного разговора

Результат: Пошаговый скрипт: приветствие → вопросы → отработка возражений → закрытие
Агент 4: Интервьюер

Генерирует вопросы для собеседования

Результат: 5 проверочных вопросов, которые отсеют неподходящих
Агент 5: Документовед

Составляет список документов для оформления

Результат: Чек-лист: паспорт, СНИЛС, медкнижка, права и т.д.
Агент 6: Онбординг-менеджер

Планирует адаптацию нового сотрудника на первую неделю

Результат: План по дням: что делать, кого знать, где учиться
Агент 7: Аналитик

Формирует итоговый отчёт по вакансии

Результат: Рекомендация: «Готово к публикации», сроки, приоритеты
